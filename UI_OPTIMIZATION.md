## 🎨 设计规范

### 颜色
- 主色：#1989fa（蓝色）
- 成功色：#07c160（绿色）
- 警告色：#ff976a（橙色）
- 危险色：#ee0a24（红色）
- 文字色：#323233
- 次要文字：#969799
- 边框色：#ebedf0
- 背景色：#f7f8fa

### 圆角
- 小圆角：8px
- 中圆角：12px
- 大圆角：16px
- 圆形：50%

### 间距
- 小间距：8px
- 中间距：12px
- 标准间距：16px
- 大间距：24px
- 超大间距：32px

### 阴影
- 轻阴影：0 2px 8px rgba(0, 0, 0, 0.04)
- 中阴影：0 4px 12px rgba(0, 0, 0, 0.08)
- 重阴影：0 8px 16px rgba(0, 0, 0, 0.12)

## 🚀 运行项目

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 访问地址
http://localhost:3000
```

## 🔧 技术栈

- Vue 3 + Composition API
- Vite 7
- Vant 4（移动端UI组件库）
- Pinia（状态管理）
- Vue Router 4
- Axios（HTTP请求）
- PostCSS + px-to-viewport（移动端适配）
- Sass/SCSS（CSS预处理器）
